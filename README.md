# Data Analytics Project

## Overview
This project focuses on analyzing a dataset using Python, SQL, and Power BI to generate meaningful business insights.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- SQL
- Power BI
- Gamma

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. SQL Analysis
5. Dashboard Creation
6. Report Generation

## Dashboard Features
- KPI Cards
- Sales Trends
- Category Analysis
- Filters & Slicers
- Interactive Visuals

## Results
- Identified key trends and patterns
- Improved understanding of business performance
- Created an easy-to-understand dashboard

## How to Run
1. Install Python libraries
2. Run Jupyter Notebook
3. Execute SQL queries
4. Open Power BI dashboard
